EC2 LAB 1 - Setup a webserver using Apache

  1. Create an EC2 instance using AWS Console with a key-pair.
  1. Change privileges to private key-pair (.pem file):
    chmod 400 key-pair.pem
  1. ssh into EC2 instance:
    ssh -i key-pair.pem ec2-user@<ip-ec2>
  1. Update operating system updates and packages:
    sudo yum update -y
  1. Install Apache (turns your EC2 into a web server):
    sudo yum install httpd -y
  1. Once installed, you should be able to:
    cd /var/www/html
  1. Anything you put in here will be available through port 80 (HTTP port). Now write a really simple html file with nano:
    sudo nano index.html
  1. Write a hello world html:
    <html><h1>Hello world!</h1></html>
  1. To exit nano, hit CTRL+X ‚Üí "Yes" ‚Üí "Enter"
  1. Run the Apache service and turn your EC2 into a web server:
    sudo service httpd start
  1. And now put chkconfig on. chkconfig on will start the httpd service if your EC2 instance accidentally reboots.
    sudo chkconfig on
  1. Go to Chrome or your web browser and put the ip of your EC2 instance, you will be able to see your new website!
