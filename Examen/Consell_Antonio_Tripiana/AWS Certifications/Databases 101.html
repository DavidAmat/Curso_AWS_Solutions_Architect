Relational Databases - RDS

What is a relational database?

Relational database is like a traditional spreadsheet (excel file):

  • Database
  • Tables
  • Row
  • Fields (Columns)

Relational databases on AWS

  • SQL Server
  • Oracle
  • MySQL Server
  • PostgreSQL
  • Aurora

RDS → Relational Database Service on AWS

RDS is an OLTP (OnLine Transaction Processing) solution .

RDS has two key features:

  • Multi-AZ - For Disaster Recovery

    Failover is automatic: If primary database fails, aws automatically change DNS pointer to another database in another AZ.

  • Read Replicas - For Performance

    aws creates a read replica so a portion of your calls go to the replica DB (lower load on first db). Limited to 5 replicas.

Data Warehousing - Redshift

Data warehousing databases use different type of architecture both from a database perspective and infrastructure ayer. Amazon's Data Warehouse Solution is Called Redshift.

Used for business intelligence. Tools like Cognos, Jaspersoft, SQL Server Reporting Services, Oracle, Hyperion, SAP NetWeaver.

Used to pull in very large and complex data sets. Usually used by management to do OLAP queries on data (such as current performance vs targets etc).

OLTP vs OLAP

OLTP - Online Transaction Processing (OLTP) differs from OLAP Online Analytics Processing (OLAP) in terms of the types of queries you will run.

OLTP Example: (ERP)

  • QUERY → select the order number 2120121
  • ANSWER → Pulls up a row of data such as Name, Date, Address to Deliver to, Delivery Status etc.

OLAP Example: (Analytics)

  • QUERY → Net Profit for EMEA and Pacific for the Digital Radio Product.
  • ANSWER → Pulls in large numbers of records such as: Sum of Radios Sold in EMEA, Sum of Radios sold in Pacific, Unit Cost of Radio in each region, Sales price of each radio, Sales price - unit cost.

Caching - ElastiCache

ElastiCache to speed up performance of existing databases (frequent identical queries).

ElastiCache is a web service that makes it easy to deploy , operate and scale an in-memory cache in the cloud. The service improves the performance of web applications by allowing you to retrieve information from fast, managed, in-memory caches, instead of relying entirely on slower disk-based databases.

ElastiCache supports to open-source in-memory caching engines:

  • Redis
  • Memcached
